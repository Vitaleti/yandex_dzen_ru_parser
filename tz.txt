Программа: Парсер новостей с dzen.ru

Важно! В первой своей версии, программа будет парсить только несколько категорий, либо вообще одну, так как некоторые
основные категори содержат в себе дополнительные подкатегории, либо имеют функционал бесконечной дозагрузки контента.
Для таких "особых" категорий прийдётся делать либо отдельный функционал, либо вовсе другой парсер

Язык: Python 3.10.2

Составные части программы:
main.py - основная программа
links.json - JSON файл со ссылками на рубрики новостей
images/... - Вложенный каталог с изображениями из новостей
data/... - Вложенный каталог с данными в формате JSON

Структура каталога images:
images/
	{rubric}/
		{datetime}/
			{uuid}.jpg

Структура каталога data:
data/
	{rubric}/
		{datetime}/
			{uuid}.json

rubric - Название рубрики по которой производился парсинг
datetime - Время запуска программы
uuid - Всемирно уникальный идентификатор, сгенерированный стандартным модулем Python

Принцип работы(на первое время): 
Программа должна открывать json файл и доставать из него ссылки.
После этого, она должна циклически обойти все ссылки из json файла и собрать следующие данные
(следующие заголовки касаются только основных ссылок, из них же после будут доставаться ссылки на конкретные новостные сводки):

title - Заголовок новости
link - Ссылка на конкретную новостную сводку(понадобиться далее при выполнении программы)
image_hyperlink - Ссылка на изображение в интернете

Далее, эти данные сохраняются в словарь. После программа переходит на страницу из переменной link,
 то есть на конкретную новостную сводку. От туда она уже берёт следующие данные:

original_link - Ссылка на полную новостную сводку
subtitles - Под заголовки новости из других похожих источников(в json файле будут записываться либо через запятую, либо ввиде массива)
subtitles_other - Ссылки из подзаголовка "Подробнее о событии". Будут храниться аналогично subtitles

После сбора блока данных, они будут дозаписываться в файл json. То есть при каждой новой итерации файл будет заново открываться.
Это хоть и снизит скорость работы программы, но не будет её сильно нагружать.
